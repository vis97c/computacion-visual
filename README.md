# Computación Visual

Ejercicios del curso de computación visual

## Estructura del proyecto

- `semana_1_1_construyendo_mundo_3d`: Ejercicios de la semana 1, lección 1
- `semana_1_2_conversion_formatos_3d`: Ejercicios de la semana 1, lección 2
- `semana_1_3_jerarquias_transformaciones`: Ejercicios de la semana 1, lección 3
- `semana_1_4_transformaciones`: Ejercicios de la semana 1, lección 4

Ejecutar un proyecto:

```bash
yarn workspace semana-1-1 run dev
```

Los otros archivos son de configuración del proyecto (Linters, formatters, etc.)
